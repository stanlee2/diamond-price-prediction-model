# 💎 Diamond Price Prediction Model

A Machine Learning-powered web application that predicts the price of diamonds based on their physical and quality-related attributes such as carat, cut, color, clarity, depth, table, and dimensions.

# 🚀 Project Overview

This project uses regression-based machine learning techniques to estimate diamond prices from structured input features. The goal is to build a predictive system that can assist jewelers, buyers, and analysts in understanding diamond valuation patterns.

The model was trained using historical diamond datasets and deployed with an interactive user interface for real-time predictions.

# 📌 Features
Predicts diamond prices instantly
Interactive UI built with Streamlit

Supports real-time inference

# Model Features and Description
Carat:   Weight of the diamond
Cut:     Quality of the cut
Color:   Diamond color grading
Clarity: Clarity measurement
Depth:   Total depth percentage
Table:   Width of top relative to widest point
X:       Length in mm
Y:       Width in mm
Z:       Depth in mm
